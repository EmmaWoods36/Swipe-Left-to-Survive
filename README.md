# Swipe Left to Survive

A turn-based dating survival RPG prototype for Amy's game.

## Current title

**Swipe Left to Survive**

Other title contenders preserved in the prototype text: **Match Made in Hell**, **Amy vs. The Algorithm**, and **The Pattern**.

**Girl, Stand Up!** is Amy's finishing move / ultimate, not the game title.

## How to run locally

Open `index.html` in a browser.

## How to publish on GitHub Pages

1. Create a repository named `Swipe-Left-to-Survive`.
2. Upload everything in this zip so `index.html` is at the repository root.
3. Go to **Settings → Pages**.
4. Set source to **Deploy from branch**.
5. Choose branch `main` and folder `/root`.
6. Save and wait for the Pages URL.
