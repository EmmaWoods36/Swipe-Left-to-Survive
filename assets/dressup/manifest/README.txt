Amy Dress-Up Game Asset Pack v1
================================

Includes a curated package of Amy paper-doll assets organized by category, plus a manifest with in-game shop pricing in Soft Life Funds.

Suggested layering order (low to high):
base -> hair -> earrings -> necklaces -> bracelets/watches -> tops/dresses/full_outfits -> outerwear -> bottoms -> shoes -> bags -> hats -> hair_ribbons -> sunglasses -> goddess/special

Notes:

- Some overlay files are isolated items; some are pre-positioned on the canvas for direct dress-up placement.
- Use asset_manifest.csv for spreadsheet-friendly setup and asset_manifest.json for game-data import.

Code-ready game data:
- manifest/amy_dressup_shop_data.json groups assets by category and includes shop visibility, ownedByDefault, renderLayer, price, and assetPath.
